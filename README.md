# Python scripts for the DigiCod module
to run on the TI-Nspire calculator

> [!WARNING]  
> Absolutely no guarantees for correctness! Submit a PR if you notice any bugs.

## How to use on the TI-Nspire
Download the latest `.tns` file from the [Releases Page](https://github.com/bb1950328/DigiCod_Nspire/releases) and transfer it to the calculator via USB.  
You need to use the "TI-Nspire™ CX CAS Student Software" or use the online [TI-Nspire™ CX II Connect](https://nspireconnect.ti.com/nsc/) app (linux users need to modify their UA)

## How to use on your PC
Technically, you will need MicroPython as that's what is running on the calculator. It should work with the standard CPython interpreter too, but many standard Python functions aren't available in MicroPython.
